A library, app, and cli for creating signed messages readable by the Algorand blockchain.

## Quickstart

```
cd cmd/cli && go run main.go  -a <CONTRACT_ADDRESS> -s "<MNEMONIC>" -d "<MESSAGE>"
```
## Run dev app.

Run dev app with vue hot reload:
```bash
make dev
```

## Build

```bash
make build
```
